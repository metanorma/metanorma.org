== Why CI/CD

Continuous Integration (CI) and Continuous Deployment (CD) automate the process
of building, testing, and deploying documents. By integrating Metanorma into
CI/CD pipelines, documents are always up-to-date and consistent, release cycles
are faster, and manual errors are reduced.

GitHub provides the most complete support through the
https://github.com/actions-mn[`actions-mn`] ecosystem of GitHub Actions.

== Organization-scale publishing

For organizations that publish many documents across many repositories,
Metanorma provides per-document releases with channel-based routing and
multi-repo aggregation into a unified portal. This feature is GitHub-only.

For the full setup guide with config schemas and checklists, see
link:/install/publication-setup/[Organization-Scale Publishing Setup].
For a step-by-step tutorial, see
link:/blog/2026-05-13-channel-based-publication/[the blog post].
