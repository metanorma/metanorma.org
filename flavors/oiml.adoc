== OIML Flavor

The OIML (International Organization of Legal Metrology) flavor for Metanorma
allows you to author OIML International Recommendations and Documents.

[cols="1,1"]
|===
| Deliverable | Description

| International Recommendation
| Normative document adopted by the OIML

| International Document
| Informative document providing guidance
|===

== Quick Start

Install the OIML flavor gem:

[source,bash]
----
gem install metanorma-oiml
----

Compile a document:

[source,bash]
----
metanorma compile -t oiml -x html document.adoc
----
