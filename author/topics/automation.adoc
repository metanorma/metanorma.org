Metanorma contains built-in mechanisms for generating content from data,
including:

* link:automation/data-to-text[Data2Text plugin: Content from YAML/JSON data]
* link:automation/yaml-to-text[Yaml2Text plugin: Content from YAML data]
* link:automation/json-to-text[Json2Text plugin: Content from JSON data]
