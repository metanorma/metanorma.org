Metanorma contains built-in mechanisms for generating content from data,
including:

* link:automation/data_to_text[Data2Text plugin: Content from YAML/JSON data]
* link:automation/yaml_to_text[Yaml2Text plugin: Content from YAML data]
* link:automation/json_to_text[Json2Text plugin: Content from JSON data]
